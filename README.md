# Dotfiles

Personal Arch Linux, ML4W Hyprland, Ghostty, Zsh, Neovim, and VS Code configuration.

## Included

- Ghostty with Tokyo Night Storm
- Zsh developer tools
- Neovim configuration
- VS Code settings
- Git defaults
- ML4W-safe Hyprland overrides
- Waybar customization files
- Installation script

## Install

Clone the repository:

    git clone https://github.com/millergitca/dotfiles.git ~/dotfiles
    cd ~/dotfiles
    ./install.sh

## Update

    cd ~/dotfiles
    git add -A
    git commit -m "Describe changes"
    git push

## Notes

This repository intentionally excludes passwords, API keys, SSH keys, tokens, and runtime state.
